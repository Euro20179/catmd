# catmd
catmd is a POSIX script that will output the given md file to stdout and prettify it,

try it on this README and see how it works

---

# usage

`catmd [-k] <file>`

# installation

```sh
git clone https://github.com/euro20179/catmd
cd catmd
chmod +x catmd
sudo cp catmd /usr/local/bin
```

# dependancies
* perl
